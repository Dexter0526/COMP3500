# 그래프 3

## 최소 신장 트리(MST)
![img.png](img/img_1.png)
- 줄여서 MST라고 함
- 신장 트리 중 비용이 최소인 트리
  - 비용 : 모든 변의 가중치를 합한 값
  - 최소 비용 신장 트리라고 함


![img.png](img/img.png)
광케이블 깔기 문제 분석
- 노드 : 아파트 빌딩
- 변 : 광선을 갈 수 있는 경로
- 변의 가중치 : 거리 = 설비 가격
- 목표
    - 모든 노드를 연결하는 변들 찾기
    - 단, 비용이 최소여야 함
  
### 신장 트리(spanning tree)
- 어떤 그래프 안에 있는 모든 노드를 연결하는 트리
- 당연히 그래프 안에 있는 변만 사용해야 함
- 신장 트리는 여럿 있을 수 있음
